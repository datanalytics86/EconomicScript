# Carpeta única de carga de muestras sanitizadas

Deja **todos** tus archivos aquí:

- `samples/dropzone/`

Puedes subir correos (`.txt/.eml/.html`) y cartolas (`.pdf/.csv`) mezclados.

## Convención sugerida de nombre

- `YYYYMMDD_banco_tipo_descripcion.ext`

Ejemplos:
- `20250115_bci_email_compra_lider.txt`
- `202501_bancoestado_cartola_tc.pdf`
- `202501_security_cartola_cc.csv`

## Importante

- Sanitiza datos sensibles (RUT, número completo de tarjeta, dirección, teléfono, etc.).
- Si puedes, agrega al inicio de cada archivo una nota con el resultado esperado (fecha, monto, comercio).
